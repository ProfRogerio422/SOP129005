# SOP129005
Este repositório reúne materiais, exemplos e recursos da disciplina Sistemas Operacionais (SOP129005). O objetivo é apoiar o aprendizado dos conceitos fundamentais de sistemas operacionais, explorando tanto a teoria quanto a prática.

Conteúdos abordados

    Estrutura e funcionamento de sistemas operacionais

    Gerenciamento de processos e threads

    Escalonamento e sincronização

    Gerenciamento de memória e sistemas de arquivos
