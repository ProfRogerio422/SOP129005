# Unidade 1 – Introdução aos Sistemas Operacionais

Objetivo: Compreender os conceitos básicos, a estrutura de um sistema operacional e suas arquiteturas.

## 📚 Conteúdos
- Definição e funções de um sistema operacional
- Estrutura de um SO (kernel, shell, utilitários)
- Tipos de arquiteturas (monolítico, microkernel, híbrido)
- Exemplos de sistemas operacionais (Windows, Linux, macOS)
